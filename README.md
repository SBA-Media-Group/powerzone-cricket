# Powerzone Cricket

Specialist batting coaching website for Powerzone Cricket, led by first-class cricketer Zobie Sharif.

## Deployment

Hosted on Netlify. Push to `main` to deploy.

## Structure

```
index.html      - Main website (single page, self-contained)
netlify.toml    - Netlify build & header config
_redirects      - SPA routing
robots.txt      - SEO crawler directives
sitemap.xml     - Sitemap for search engines
```

## Setup

1. Push this repo to GitHub
2. Connect the repo to Netlify
3. Set custom domain: `powerzonecricket.com`
4. Netlify will auto-deploy on every push to `main`
